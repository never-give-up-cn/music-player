# 音乐播放器

一个简洁美观的本地音乐播放器，双击 `player.html` 即可在浏览器中播放音乐。

## 使用方式

1. 下载 MP3 文件放入 `音乐\` 文件夹
2. 按 `songs.js` 中的 `file` 字段命名文件
3. 双击 `player.html` 在浏览器中打开
4. 点击播放按钮或按 `空格键` 开始播放

## 功能

- 播放/暂停（空格键）
- 上一首/下一首（← / → 方向键）
- 进度拖拽
- 音量调节（↑ / ↓ 方向键）
- 播放模式：顺序 / 单曲循环 / 随机
- 播放列表侧栏
- 深色毛玻璃 UI
- 响应式设计（支持移动端）

## 技术栈

纯前端 HTML5 + CSS3 + JavaScript，无任何外部依赖。
